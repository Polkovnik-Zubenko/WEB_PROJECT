Прикрепите архив с расширение rar, в котором буду содержаться тесты для задачи.
Сначала в архиве должна быть папка под названием tests, а в ней должны содержаться
файлы без расширения (то есть изначальное расширение txt, после просто удаляем расширение) и файлы с расширением "a". Файл без расширения предназначен
для входных данных, а файл с расширение "a" предназначен для результата. У каждого файла
должна быть пара, то есть если существует файл "1", то должен быть и файл "1.a".
Файлы нужно называть от одного 1 и до бесконечности(в зависимости сколько имеется тестов).
